# Gondli

Gondli is a modern web-based sports, wellness, and service booking platform designed to connect users with service providers through a seamless digital experience. The platform allows customers to discover services, book appointments, manage profiles, and complete payments, while business owners can efficiently manage bookings and daily operations through a dedicated dashboard.

## Features

### User Features
- User registration and login
- Secure authentication system
- Browse available services
- Smart search by service, therapist, or location
- View service details and pricing
- Book appointments online
- Profile and account management
- Responsive design for desktop and mobile devices

### Business Owner Features
- Booking management dashboard
- View upcoming and active reservations
- Accept or cancel bookings
- Daily and list booking views
- Refund workflow support
- Manage schedules and appointments

## Technologies Used

### Frontend
- Next.js
- TypeScript

- Redux Toolkit
- SCSS / CSS

### Backend
